The HyperSDK aims to provide good defaults, making it simple to build high performance blockchains, while enabling customization at each level of the stack. In this chapter we will explore some of these concepts.

You can see the code snippets in this course, but you can also browse through the HyperSDK Starter Kit to see the code in the context of a full project.
