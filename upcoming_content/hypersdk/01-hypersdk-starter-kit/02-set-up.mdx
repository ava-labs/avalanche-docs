The HyperSDK Starter Kit contains everything you need to get started quickly with the HyperSDK.

In this course we will run the HyperSDK Starter Kit in a hosted environment on Github. This is the most quick way to get started. You can alternatively run the HyperSDK Starter Kit locally on your machine with docker and docker compose.

<Steps>
<Step>

### Clone the HyperSDK Starter Kit Repo

```bash
git clone https://github.com/ava-labs/hypersdk-starter-kit.git
```

</Step>
<Step>

### Launch this example

Open the terminal with `` Ctrl + ` ``. Now run the following command to start the devnet, faucet, and frontend:

```bash
docker compose up -d --build devnet faucet frontend
```

This may take 5 minutes to download dependencies.

</Step>
</Steps>

You can wind everything down after the exercises with `docker compose down`.
