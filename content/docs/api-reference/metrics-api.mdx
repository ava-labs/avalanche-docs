The API allows clients to get statistics about a node's health and performance.

<Callout title="Note">
This API set is for a specific node, it is unavailable on the [public server](/docs/tooling/rpc-providers).
</Callout>

Endpoint[​](#endpoint "Direct link to heading")
-----------------------------------------------

```
/ext/metrics
```

Usage[​](#usage "Direct link to heading")
-----------------------------------------

To get the node metrics:

```
curl -X POST 127.0.0.1:9650/ext/metrics
```

Format[​](#format "Direct link to heading")
-------------------------------------------

This API produces Prometheus compatible metrics. See [here](https://github.com/prometheus/docs/blob/master/content/docs/instrumenting/exposition_formats.md) for information on Prometheus' formatting.

[Here](/docs/nodes/maintain/monitoring) is a tutorial that shows how to set up Prometheus and Grafana to monitor AvalancheGo node using the Metrics API.
