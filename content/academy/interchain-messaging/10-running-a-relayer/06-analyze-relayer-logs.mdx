To analyze the relayer activity you can check the relayer logs. If you're using the Avalanche CLI, you can access them with the following command:

```bash
avalanche teleporter relayer logs
```

Alternatively the JSON logs can be located under "~/.avalanche-cli/runs/awm-relayer.log". Let's take a look at them:

```bash
code ~/.avalanche-cli/runs/awm-relayer.log 
```
